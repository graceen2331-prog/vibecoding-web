# Vibe Coding MVP 快速开发路线图 (10 天)

**开发模式**：AI 驱动快速开发 + 内容分离  
**项目周期**：10 天（从 2026年1月29日 开始）  
**目标上线**：2026年2月8日  
**开发目标**：完成平台核心功能（登陆、编辑器、课程、成就、分享）
**内容计划**：并行创作（不阻塞平台上线）

---

## 🚀 快速开发策略

### 核心原则
- ✅ **功能开发优先**（本周期重点）：构建可运行的完整平台
- ⏳ **内容创作分离**（并行推进）：课程讲义、视频、代码框架可独立进行
- 🤖 **AI 全力加速**：所有代码由 AI 生成，开发者主要进行集成和测试
- ⚡ **最小化 MVP**：剔除非必要功能，只保留核心流程

### 时间对比
| 传统方式 | Vibe 模式 |
|--------|---------|
| 6 周 | **10 天** |
| 173 小时 | **~50 小时** |
| 完整课程 + 功能 | 功能✅ + 课程⏳ |

## 🔧 技术快速参考

### 使用 AI 加速的命令示例

```bash
# 1. 让 AI 生成 React 组件
# 在 Cursor 中输入：
# "Generate a React login page component with email input, using Tailwind CSS.
#  Call the API at POST /api/auth/login with the email."

# 2. 让 AI 生成 Express API
# "Generate an Express.js route to handle GET /api/courses
#  that returns a list of courses from the database using Prisma."

# 3. 让 AI 生成 Prisma 模型
# "Generate Prisma schema for a course learning platform with 
#  users, courses, user_progress, and achievements tables."
```

### 核心依赖清单（已安装）
```
前端：React 18 + Vite + Tailwind + React Router
后端：Express + Prisma + PostgreSQL
编辑器：WebContainers + Monaco Editor
认证：JWT + Magic Link
数据库：Supabase (PostgreSQL)
```
